# My Todo (React + Vite)

このリポジトリには、Vite を使ったシンプルな React Todo アプリの最小実装が含まれます。

## 前提
- Node.js 16+
- npm または yarn

## セットアップ

```bash
# 依存をインストール
npm install

# 開発サーバーを起動
npm run dev
```

ブラウザで http://localhost:5173 を開くとアプリが動作します（Viteのデフォルトポート）。

## ビルド

```bash
npm run build
npm run preview
```
